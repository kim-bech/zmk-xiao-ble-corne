# zmk-corne-xiao-ble
This is a fork of the zmk repo for the Ali Express corne clone with a seeed studio xiao ble board. 
The pcb used says "abomination v.1.2" on it.
Unfortunately a lot of sellers on AE don't have a clue what firmware their product is using, so it took me a while to find the correct repo.
I hope this can help others searching for the correct firmware and I will try to work on it to get it up to date with latest zmk features
